- Esta aplicación Springboot se generó en war por lo que se recomienda abrir desde un eclipse y ejecutarlo.
- El nombre de la base de datos mongo que usa es "miaguila".
- Este proyecto se conecta a una base de datos local mongo.

#Instrucciones GUI
Presiona el botón de enviar y recuperas un registro completo de la base de datos y lo pinta. Esto se hace através de un WebSocket.
